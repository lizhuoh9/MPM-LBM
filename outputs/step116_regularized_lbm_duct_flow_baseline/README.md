# Step116 Regularized LBM Duct Flow Baseline Artifacts

These files are simulation-backed LBM-only Step116 artifacts. They are bounded
probes, not full 48^3/96^3 long-window validation.

Each row records the requested matrix name and the actual committed execution
size in `run_metadata.json` and `finite_stability_report.json`.

Do not treat these artifacts as Fluent validation, Figure 29.3 parity, or full
FSI evidence.
