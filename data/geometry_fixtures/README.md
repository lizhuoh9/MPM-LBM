# Step 20 Geometry Fixtures

Small synthetic fixtures for the mesh / voxel geometry import pipeline.

These files are intentionally small and deterministic:

- `voxel_sphere.npy`: 32 x 32 x 32 synthetic occupancy sphere.
- `voxel_sphere_metadata.json`: metadata for the voxel fixture.
- `cube.obj`: small watertight cube mesh with 8 vertices and 12 triangles.
- `ellipsoid_proxy.obj`: small synthetic ellipsoid-like watertight mesh.

These fixtures are not real squid geometry, not anatomical validation, and not production mesh repair input.
