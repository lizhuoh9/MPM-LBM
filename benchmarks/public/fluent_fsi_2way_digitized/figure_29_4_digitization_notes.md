# Figure 29.4 Digitization Notes

This folder contains a derived approximate numeric reference from the public Ansys Fluent tutorial Figure 29.4. It does not contain an official Fluent case, mesh, journal, data file, or copied official image.

The first Step107 curve was manually read from the public plot and linearly interpolated to the official 50-step transient grid: `time_s = 0.0000, 0.0005, ..., 0.0250`.

The uncertainty is intentionally conservative at `2.5e-5 m` on every row. Treat this as a public-plot comparison target only, not as raw Fluent data and not as validation evidence.
